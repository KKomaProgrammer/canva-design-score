Canva Design Score 설치

1. Chrome에서 chrome://extensions 또는 Edge에서 edge://extensions를 엽니다.
2. 개발자 모드를 켭니다.
3. 압축해제된 확장 프로그램을 로드하고 이 폴더를 선택합니다.
4. Canva 디자인 편집 화면을 열고 확장 아이콘을 누릅니다.
5. Cloudflare Pages의 /api/analyze 주소와 Access Token을 저장합니다.
6. 기본 GPT-5.6 Luna로 분석하거나, 필요할 때 Terra를 선택합니다.

주의: 캡처 중에는 해당 Canva 탭을 다른 창으로 가리지 말고 활성 상태로 유지하세요.
브라우저 설정·개발자 도구 같은 내부 페이지에서는 분석이 자동으로 건너뛰어집니다.
